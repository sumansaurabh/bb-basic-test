# Einfache Next.js Website

Eine einfache Next.js-Anwendung mit TypeScript und Tailwind CSS.

## Erste Schritte

### Entwicklung
Starten Sie den Entwicklungsserver:
```bash
pnpm dev
```
Öffnen Sie [http://localhost:3000](http://localhost:3000), um die Website anzuzeigen.

### Build
Erstellen Sie einen Produktions-Build:
```bash
pnpm build
```

### Produktion
Starten Sie den Produktionsserver:
```bash
pnpm start
```

### Linting
Führen Sie ESLint aus:
```bash
pnpm lint
```

## Projektstruktur
- `src/app/page.tsx` - Hauptseiten-Komponente
- `src/app/layout.tsx` - Root-Layout-Komponente
- `src/app/globals.css` - Globale Styles
- `public/` - Statische Assets
