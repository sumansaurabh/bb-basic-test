# መሰረታዊ Next.js ድህረ ገጽ

በ TypeScript እና Tailwind CSS የሆነ ቀላል Next.js መተግበሪያ።

## መጀመር

### ለልምድ
የልምድ አገልጋይ ለማስኬድ፡-
```bash
pnpm dev
```
ድህረ ገጁን ለማየት [http://localhost:3000](http://localhost:3000) ይክፈቱ።

### ለመስራት
የምርት ስራ ለመስራት፡-
```bash
pnpm build
```

### ለምርት
የምርት አገልጋይ ለማስኬድ፡-
```bash
pnpm start
```

### ለመፈተሽ
ESLint ለማስኬድ፡-
```bash
pnpm lint
```

## የፕሮጀክት መዋቅር
- `src/app/page.tsx` - ዋና የገጽ አካል
- `src/app/layout.tsx` - የሥር አቀራር አካል
- `src/app/globals.css` - የዓለም አቀራር
- `public/` - የማያለስ ንጥረ ነገሮች
