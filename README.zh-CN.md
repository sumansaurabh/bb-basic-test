# 基础 Next.js 网站

一个简单的 Next.js 应用程序，使用 TypeScript 和 Tailwind CSS。

## 快速开始

### 开发模式
运行开发服务器：
```bash
pnpm dev
```
在浏览器中打开 [http://localhost:3000](http://localhost:3000) 查看网站。

### 构建
创建生产环境构建：
```bash
pnpm build
```

### 生产模式
启动生产服务器：
```bash
pnpm start
```

### 代码检查
运行 ESLint：
```bash
pnpm lint
```

## 项目结构
- `src/app/page.tsx` - 主页面组件
- `src/app/layout.tsx` - 根布局组件
- `src/app/globals.css` - 全局样式
- `public/` - 静态资源
