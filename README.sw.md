# Tovuti ya Msingi ya Next.js

Programu rahisi ya Next.js ikiwa na TypeScript na Tailwind CSS.

## Kuanza

### Maendeleo
Endesha seva ya maendeleo:
```bash
pnpm dev
```
Fungua [http://localhost:3000](http://localhost:3000) kuona tovuti.

### Kujenga
Unda ujenzi wa uzalishaji:
```bash
pnpm build
```

### Uzalishaji
Anzisha seva ya uzalishaji:
```bash
pnpm start
```

### Ukaguzi wa Msimbo
Endesha ESLint:
```bash
pnpm lint
```

## Muundo wa Mradi
- `src/app/page.tsx` - Kipengee cha ukurasa mkuu
- `src/app/layout.tsx` - Kipengee cha muundo wa mzizi
- `src/app/globals.css` - Mitindo ya kimataifa
- `public/` - Mali thabiti
