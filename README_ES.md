# Sitio Web Básico de Next.js

Una aplicación simple de Next.js con TypeScript y Tailwind CSS.

## Comenzando

### Desarrollo
Ejecutar el servidor de desarrollo:
```bash
pnpm dev
```
Abrir [http://localhost:3000](http://localhost:3000) para ver el sitio web.

### Compilación
Crear una compilación de producción:
```bash
pnpm build
```

### Producción
Iniciar el servidor de producción:
```bash
pnpm start
```

### Linting
Ejecutar ESLint:
```bash
pnpm lint
```

## Estructura del Proyecto
- `src/app/page.tsx` - Componente de página principal
- `src/app/layout.tsx` - Componente de diseño raíz
- `src/app/globals.css` - Estilos globales
- `public/` - Recursos estáticos
