# Site Web Next.js de Base

Une application Next.js simple avec TypeScript et Tailwind CSS.

## Démarrage

### Développement
Exécutez le serveur de développement :
```bash
pnpm dev
```
Ouvrez [http://localhost:3000](http://localhost:3000) pour voir le site web.

### Construction
Créez une version de production :
```bash
pnpm build
```

### Production
Démarrez le serveur de production :
```bash
pnpm start
```

### Linting
Exécutez ESLint :
```bash
pnpm lint
```

## Structure du Projet
- `src/app/page.tsx` - Composant de la page principale
- `src/app/layout.tsx` - Composant de mise en page racine
- `src/app/globals.css` - Styles globaux
- `public/` - Ressources statiques
