# मूलभूत Next.js वेबसाइट

TypeScript आणि Tailwind CSS सह एक साधा Next.js अनुप्रयोग.

## सुरुवात करणे

### विकास
विकास सर्व्हर चालवा:
```bash
pnpm dev
```
वेबसाइट पाहण्यासाठी [http://localhost:3000](http://localhost:3000) उघडा.

### बिल्ड
उत्पादन बिल्ड तयार करा:
```bash
pnpm build
```

### उत्पादन
उत्पादन सर्व्हर सुरू करा:
```bash
pnpm start
```

### लिंटिंग
ESLint चालवा:
```bash
pnpm lint
```

## प्रकल्प रचना
- `src/app/page.tsx` - मुख्य पृष्ठ घटक
- `src/app/layout.tsx` - मूळ लेआउट घटक
- `src/app/globals.css` - जागतिक शैली
- `public/` - स्थिर मालमत्ता
