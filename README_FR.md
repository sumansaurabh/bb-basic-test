# Site Web Next.js de Base

Une application Next.js simple avec TypeScript et Tailwind CSS.

## Commencer

### Développement
Exécuter le serveur de développement :
```bash
pnpm dev
```
Ouvrir [http://localhost:3000](http://localhost:3000) pour voir le site web.

### Build
Créer un build de production :
```bash
pnpm build
```

### Production
Démarrer le serveur de production :
```bash
pnpm start
```

### Linting
Exécuter ESLint :
```bash
pnpm lint
```

## Structure du Projet
- `src/app/page.tsx` - Composant de page principale
- `src/app/layout.tsx` - Composant de mise en page racine
- `src/app/globals.css` - Styles globaux
- `public/` - Ressources statiques
