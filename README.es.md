# Sitio Web Básico de Next.js

Una aplicación simple de Next.js con TypeScript y Tailwind CSS.

## Comenzando

### Desarrollo
Ejecuta el servidor de desarrollo:
```bash
pnpm dev
```
Abre [http://localhost:3000](http://localhost:3000) para ver el sitio web.

### Construcción
Crea una compilación de producción:
```bash
pnpm build
```

### Producción
Inicia el servidor de producción:
```bash
pnpm start
```

### Linting
Ejecuta ESLint:
```bash
pnpm lint
```

## Estructura del Proyecto
- `src/app/page.tsx` - Componente de página principal
- `src/app/layout.tsx` - Componente de diseño raíz
- `src/app/globals.css` - Estilos globales
- `public/` - Recursos estáticos
