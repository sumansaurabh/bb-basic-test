# அடிப்படை Next.js வலைத்தளம்

TypeScript மற்றும் Tailwind CSS உடன் கூடிய எளிய Next.js பயன்பாடு.

## தொடங்குதல்

### மேம்பாடு
மேம்பாட்டு சேவையகத்தை இயக்கவும்:
```bash
pnpm dev
```
வலைத்தளத்தைப் பார்க்க [http://localhost:3000](http://localhost:3000) ஐத் திறக்கவும்.

### உருவாக்கம்
உற்பத்தி உருவாக்கத்தை உருவாக்கவும்:
```bash
pnpm build
```

### உற்பத்தி
உற்பத்தி சேவையகத்தைத் தொடங்கவும்:
```bash
pnpm start
```

### குறியீடு சரிபார்ப்பு
ESLint ஐ இயக்கவும்:
```bash
pnpm lint
```

## திட்ட அமைப்பு
- `src/app/page.tsx` - முதன்மை பக்க கூறு
- `src/app/layout.tsx` - மூல அமைப்பு கூறு
- `src/app/globals.css` - உலகளாவிய பாணிகள்
- `public/` - நிலையான சொத்துக்கள்
